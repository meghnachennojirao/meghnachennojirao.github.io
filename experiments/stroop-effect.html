Color And Numerical Stroop Test

Measure how quickly you can answer the task-relevant feature while ignoring a competing feature: ink color, digit value, or number-word meaning.

Rules

  • Color trials: name the ink color, ignore the word.
  • Digit trials: report how many digits are shown, ignore digit value.
  • Word-number trials: report how many words are shown, ignore word meaning.
  • Respond as fast as you can without guessing wildly.
