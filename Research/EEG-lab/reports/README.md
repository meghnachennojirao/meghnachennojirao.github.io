# EEG Report Practice Material

This folder is reserved for curated EEG report practice material.

The app should not commit raw clinical EEG archives or large EDF recordings here. Store only small, de-identified educational excerpts, source manifests, and derived teaching annotations.

## Intended Sources

- Temple University Hospital EEG Corpus for physician report excerpts after access approval.
- PhysioNet CHB-MIT, Siena, EEG Motor Movement/Imagery, and CAP Sleep datasets for open-access EEG signals, annotations, and metadata-backed exercises.

## Report Excerpt Rules

Each report excerpt should be:

1. De-identified.
2. Short enough for mobile reading.
3. Linked to a source manifest entry.
4. Paired with a teaching objective.
5. Paired with a common interpretation error.

## Practice Format

Early lessons should show only one line:

```text
Referral: episodes of staring.
```

Intermediate lessons can show description and impression separately:

```text
Description: posterior dominant rhythm is present and symmetric.
Impression: normal awake EEG.
```

Advanced lessons can ask the learner to choose or write the impression:

```text
Question: Which impression best answers the referral question?
```
